# User Search & Filter System

## 📌 Description
This project is a Python-based application that fetches user data from a REST API and allows users to interactively search and filter the data.

## 🚀 Features
- Fetch all users from API
- Display all user names
- Search users by name
- Filter users by city
- Interactive menu-driven system
- Handles cases where no results are found

## 🛠 Technologies Used
- Python
- Requests library
- REST API
- JSON data handling

## ▶️ How to Run
1. Run the Python file
2. Choose an option from the menu:
   - Show all users
   - Search by name
   - Filter by city
   - Exit
3. Enter required input
4. View results

## 📊 Example Use Cases
- View all users in the system
- Search for a specific user by name
- Find users from a particular city
